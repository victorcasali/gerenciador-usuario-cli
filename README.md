# Gerenciador de Usuários (CLI)

Um projeto em Python feito para praticar lógica de programação e manipulação de arquivos. O objetivo é gerenciar usuários via linha de comando (CLI) de forma simples e organizada.

 ## **Funcionalidades** :
- Cadastrar usuário (Nome, Idade, Sexo)
- Listar usuários cadastrados
- Armazenamento em arquivo `.txt`
- Menu interativo em loop
- Tratamento de erros (valores inválidos, interrupções, etc.):
 

## **Estrutura do Projeto**
```
Gerenciador de Usuário-CLI/
├── dados.txt               # Arquivo onde os usuários são armazenados
├── interface.py            # Funções de interface do usuário
├── arquivo.py              # Funções de leitura e escrita
├── menu.py                 # Funções do menu principal
└── Sistema_Gerenciador.py  # Programa principal
```

 ## **Como Executar**: 
1. Certifique-se de ter o **Python 3** instalado.
2. Abra o terminal na pasta do projeto.
3. Execute:
```
python Sistema_Gerenciador.py
```
